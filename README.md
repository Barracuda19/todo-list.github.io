# To-Do List App | Aplicație To-Do List

## Descriere / Description

Aceasta este o aplicație To-Do List realizată pentru portofoliu, modernă și complet funcțională.  
Permite adăugarea, editarea, ștergerea și bifarea task-urilor, cu persistenta în **localStorage**.

This is a modern, fully functional To-Do List app created for portfolio purposes.  
It allows adding, editing, deleting, and marking tasks as done, with persistence using **localStorage**.

---

## Funcționalități / Features

- Adaugă / Editează / Șterge task-uri | Add / Edit / Delete tasks
- Click pentru Done → linie prin text | Click to mark as Done → line-through text
- Dublu-click pentru edit inline | Double-click to edit inline
- Filtre: All / Active / Completed | Filters: All / Active / Completed
- Empty state vizibil când lista este goală | Empty state visible when list is empty
- Animații subtile la Done și hover | Subtle animations on Done and hover
- Culori personalizabile pe butoane | Customizable button colors

---

## Tehnologii / Technologies

- HTML5
- CSS3 (Flexbox, hover, tranziții)
- JavaScript (DOM, localStorage, Event Listeners)

---

## Cum rulezi / How to Run

1. Descarcă sau clonează repository-ul | Download or clone the repository
2. Deschide `index.html` în browser | Open `index.html` in your browser
3. Toate task-urile se salvează automat în localStorage | All tasks are automatically saved in localStorage
